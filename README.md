## Интерфейс

<img width="412" height="310" alt="image" src="https://github.com/user-attachments/assets/93b87c5c-ed06-455a-baea-4d16053fec8b" />

+ Auto_Personnel – страница в которой хранится информация о персонале
+ Auto – страница в которой хранится информация о транспорте
+ Journal – страница в которой хранится информация о времени выезда-приезда
+ Routes – страница в которой хранится информация о маршруте
+ Login – вход пользователя
+ Sign_Up – регистрация пользователя
